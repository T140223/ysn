# ysn
# 🔒 匿名抽号系统

一个注重隐私保护的随机抽号工具，所有参与者均以**匿名代号**（如 `ID_001` / `参与者_1`）标识，不包含任何真实姓名或学号信息。支持**不重复抽取**（无放回）和**可重复抽取**（有放回），并自动记录抽取历史。

## ✨ 功能特性

- 🕶️ **完全匿名**：所有数据均以代号形式存储和展示，无个人隐私泄露风险。
- 🎲 **两种抽号模式**
  - **不重复模式**：已抽中的代号从候选池中移除，直到全部抽完（适合奖品不重复发放）。
  - **可重复模式**：每次抽取独立，同一代号可被多次抽中（适合模拟随机抽选）。
- 🔢 **批量抽取**：支持一次抽取多人（上限200人，可自定义数量）。
- 📜 **抽号历史**：自动记录每次抽取的详细结果和时间，支持一键清空。
- 🎨 **现代 UI**：毛玻璃效果、渐变色按钮、响应式布局，手机电脑均友好适配。
- ⚡ **纯前端实现**：所有数据保存在代码中，无需后端，隐私安全。

## 👥 匿名池说明

系统内置 **67 个匿名代号**，格式为：
- 代号：`ID_001` ~ `ID_067`
- 显示名称：`参与者_1` ~ `参与者_67`

> 这些代号与原始学生数据**顺序一一对应**，但**完全不包含真实学号或姓名**，仅用于随机抽取展示。

## 🚀 如何使用

1. **下载/保存文件**  
   将提供的完整 HTML 代码复制到一个文本编辑器中，保存为 `anonymous_draw.html`（任意以 `.html` 结尾的文件名）。

2. **打开系统**  
   双击该 `.html` 文件，系统会自动在默认浏览器中运行。

3. **配置抽号参数**  
   - **抽取数量**：设置每次抽取的人数（例如 `1`、`3`、`5` 等）。  
   - **抽号模式**：点击切换“可重复”或“不重复”模式。

4. **开始抽号**  
   点击 **🎲 开始抽取代号** 按钮，结果会显示在中央区域，并自动记录到历史列表中。

5. **管理抽号池**  
   - **重置抽号池**：在不重复模式下，将已抽中的代号全部放回，恢复到完整名单。  
   - **清空历史**：仅清除下方的历史记录，不影响当前抽取池。

## 🛠️ 自定义修改（更改代号数量或名称）

如果您需要修改匿名池中的代号数量或显示名称，请按照以下步骤操作：

1. 用任何文本编辑器（记事本、VS Code 等）打开 `anonymous_draw.html` 文件。
2. 找到 JavaScript 代码中的 `ANONYMOUS_ITEMS` 生成循环（大约在 `// ---------- ★ 匿名数据 ... ★ ----------` 注释下方）：
   ```javascript
   const ANONYMOUS_ITEMS = [];
   for (let i = 1; i <= 67; i++) {
       ANONYMOUS_ITEMS.push({
           id: `ID_${String(i).padStart(3, '0')}`,
           name: `参与者_${i}`
       });
   }
## 修改循环次数（如改为 i <= 100）即可增加总数；修改 id 或 name 的格式可自定义显示内容。
## 如果需要使用固定自定义代号列表，也可以直接写成数组形式：
const ANONYMOUS_ITEMS = [
    { id: "A001", name: "匿名者1" },
    { id: "B002", name: "匿名者2" },
    // ...
];
## 保存文件，刷新浏览器即可生效。
注意：修改后总人数会自动更新，无需额外配置。
❓ 常见问题
Q：为什么“不重复”模式下抽了几次后提示没有剩余代号？
A：这是因为所有代号都已被抽过一次。点击 重置抽号池 即可重新开始。
Q：可以一次抽取超过200个代号吗？
A：出于性能考虑，可重复模式单次上限为200；不重复模式上限为当前剩余数量。
Q：历史记录会保存到下次打开页面吗？
A：不会。每次刷新或重新打开页面，系统都会重置状态（池子恢复完整，历史清空）。若需持久化，可将历史数据导出（可自行扩展功能）。
Q：我能否恢复真实的学号和姓名？
A：可以。您只需将 ANONYMOUS_ITEMS 数组替换为原始数据对象即可，但请注意隐私保护责任。
